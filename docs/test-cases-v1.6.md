## 一、测试前准备

| 项 | 说明 |
|----|------|
| 启动 | `cd DeepSleep && npm run dev` |
| 清空数据 | DevTools → Application → Local Storage → 清除站点数据（每轮大场景建议用隐身窗口） |
| 测试邮箱 | Mock 即可，如 `partner@firm.com`、`manager@firm.com`、`incharge@firm.com` |
| 持久化验证 | 关键用例执行后 **F5 刷新**，确认数据仍在 |
